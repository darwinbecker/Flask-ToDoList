Die ToDoList.py Datei, enthält den gesamten Python Programmcode und befindet sich in keinem Unterverzeichnis.
Die ToDo-Datenbank.db Datei besitzt 2 Tabellen, die eine Tabelle speichert alle 'NEW' Einträge und die andere Tabelle speichert alle 'FINISHED' Einträge. Diese Datei befindet sich ebenfalls in keinem Unterverzeichnis.

Die HTML Datei befindet sich im templates Ordner und ist die einzige HTML Datei. Alle Funktionen bzw. Variablen werden auf dieser Seite angezeigt.

Das CSS stylesheet befindet sich wie vorgeschrieben in einem static Ordner. Aus diesem Ordner kann Flask das stylesheet erfolgreich laden.

Im Verzeichnis 'venv' befinden sich die benötigten frameworks bzw. libraries.

Das UML Diagram liegt ebenfalls in keinem Unterverzeichnis und ist eine PDF Datei.
